When rake_generate produces errors like this:

```
Build Warning: Layout 'nil' requested in blog/tags/git/atom.xml does not exist.
```

Fix by replacing:

```
layout: nil
```

with:

```
layout: null
```

in files:

- `source/atom.xml`
- `source/robots.txt`
- `source/_includes/custom/category_feed.xml`
