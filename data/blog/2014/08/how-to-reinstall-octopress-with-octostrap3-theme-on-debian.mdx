### Reinstall Octopress

```
cd [path-to-repos]
git clone git@github.com:[your-github-pages-repo.git] octopress
cd octopress
git checkout source
rake setup_github_pages
```

### Re-enable theme submodule

```
git submodule init
git submodule update
cd .themes/octostrap3
git checkout master
```
